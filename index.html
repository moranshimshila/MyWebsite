    Question 1

    In my opinion, the 3 main differences between web1, web2 and web3 are:
    1) Compared with Web 2.0 and Web 3.0, Web 1.0 was an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.
    There was no flow or communication between the consumer and the producer of the information, static pages.

    2) Compared with Web 1.0, Web 2.0 encourages participation, collaboration, and information sharing. Examples of Web 2.0 applications are Youtube, Wiki, Flickr, Facebook.

    3) Compared with Web 2.0, In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users.
    Web 3.0 consists natural language search, recommendation agents (AI), personalized search, Semantic Web, 3D Graphics.
    In addition, Content is accessible by multiple applications, every device is connected to the web, the services can be used everywhere.


    Question 2

    Example of a Use- Case:
    Search of the query "Ben Gurion"

    At web 1.0:
    Not possible.
    There were no search engines on the Internet because there was no technology to do so.
    There were sites where you could search for the information you requested.

    At web 2.0: Possible.
    It was possible to run queries on search engines in the Internet such as Google.
    The result was all the content that exists in the question, but not necessarily the answer itself.

    At web 3.0: Possible with more advanced options.
    Nowadays, Search engines try to return the most relevant result to the user.The data is being analyzed
    There is a technology of natural language processing and you can convert a human voice into text instead of typing the query on the keyboard.
    If the Google search engine knows that I am a student at Ben-Gurion University, the university site will be the first result that will return from the query before the first prime minister of Israel.
    The reason for this is because the search is personalized and based on based preferences.
    Each person will receive query result with different information that is specific to him.
